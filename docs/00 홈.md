# 멀티 LLM 리서치 에이전트 — 설계서

여러 LLM(Claude·GPT·Gemini)이 동일한 주제를 병렬 조사하고 상호 토론한 뒤,
종합된 결과를 PPT / Word / Excel 보고서로 생성하는 Streamlit 프로그램의 설계 문서.

> 프로젝트 경로: `C:\ANFILT_AI\Research Agent`
> 실행: `streamlit run app.py`

## 문서 목차

- [[01 아키텍처]] — 전체 구조, 모듈 맵, 데이터 흐름
- [[02 파이프라인 설계]] — 조사→토론→채점→종합 로직, 프롬프트, 보고서 스키마
- [[03 LLM 프로바이더]] — 3사 연동 상세, 모델, 알려진 이슈와 해결책
- [[04 레퍼런스 시스템]] — URL 수집 / 자동 탐색 / 파일 첨부
- [[05 보고서 생성]] — 분량 계획, PPT 3종 레이아웃, Word/Excel 빌더
- [[06 PPT 디자인 엔진]] — Gamma·Canva·Google Slides 연동과 키 발급 절차
- [[07 설치와 실행]] — 환경 구성, 환경변수, 실행 방법
- [[08 작업 이력]] — 날짜별 개발 내역과 의사결정 기록
- [[09 향후 고도화 백로그]] — 다음 단계 아이디어와 미해결 항목
- [[10 배포와 홈페이지 연동]] — Streamlit Cloud 배포 + 접근정책(내부용→비번게이트 공개)
- [[11 원커맨드 배포]] — "배포해줘" 한 마디로 전체 자동 배포 (git push = 배포)
- [[12 홈페이지 제품 페이지와 상담 관리]] — 제품 페이지 분리(/product/) + 상담 관리자 화면(/admin/)
- [[13 지식 볼트와 온톨로지]] — 조사 결과 영속화 + 사용자별 ESG 온톨로지(Obsidian) 축적·주입
