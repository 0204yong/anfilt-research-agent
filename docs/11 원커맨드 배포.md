# 11 원커맨드 배포 (자동 배포 체계)

사용자 요구(2026-07-04): "일부만 수정해도 배포 명령 한 번이면 내 개입 없이 전부 배포돼야 한다."
→ 두 배포 대상을 모두 **git push = 배포** 구조로 전환해 해결.

## 아키텍처

```
[사용자] "수정해줘" → [Claude] 코드 수정
[사용자] "배포해줘" → [Claude] 시크릿 검사 → 커밋 → push ×2
                          ├─ anfilt-homepage  → Netlify 자동 빌드·배포 (~1분)
                          └─ anfilt-research-agent → Streamlit 자동 재배포 (~1-3분)
                      → [Claude] 라이브 검증 → 보고
```

## 구축 내역 (2026-07-04)

### 홈페이지: Netlify Drop(수동) → GitHub 연동(자동)으로 전환
1. `Company_Homepage`를 git 저장소화 → GitHub `0204yong/anfilt-homepage` (비공개)
   - `.gitignore`: `dist/`(빌드 산출물), `.obsidian/`, dist.zip 제외
   - **`netlify.toml`** 추가 — 소스의 진실을 `concept-c.html` 하나로:
     ```toml
     [build]
       publish = "dist"
       command = "mkdir -p dist && cp concept-c.html dist/index.html && cp -r assets dist/assets"
     ```
2. Netlify 사이트(기존 anfilt-homepage, URL 불변)에 저장소 연결
   - 경로: Project configuration → Build & deploy → **Link repository**
   - GitHub OAuth 승인 + Netlify GitHub App 설치(All repositories)는 **사용자가 클릭**
     (OAuth 권한 부여는 에이전트 대행 불가 규정). 나머지 위저드는 에이전트가 브라우저로 진행.
   - netlify.toml이 3단계 설정에 자동 반영됨 → Deploy 클릭으로 연결 완료.
3. 검증: 첫 자동 빌드 정상(assets 포함), **e2e 증명** — HTML 주석 마커 추가 →
   push만 했는데 라이브에 자동 반영 확인.

### 앱: 원래부터 자동 (Streamlit Cloud가 main push를 감시)
- 추가 조치 없음. push하면 1~3분 내 재배포.
- 단, **Secrets(API 키)와 Sharing(초대 목록)은 git 밖** — Streamlit 설정 UI에서만 변경.

### 배포 명령 체계
- **`/deploy` 스킬**: `Research Agent/.claude/skills/deploy/SKILL.md`
  — "배포해줘" 한마디로 Claude가 전 절차(시크릿 검사→커밋→push→검증→보고) 수행
- **수동 스크립트**: `C:\ANFILT_AI\deploy.ps1 -m "메시지"` (Claude 없이도 동일 동작)

## 시크릿 사전 검사 (교훈 포함)

- push 전 추적 파일 전체를 **실키 형태 정규식**으로 스캔:
  `sk-ant-api[0-9]{2}-…{20,}`, `AIzaSy…{30,}`, `AQ.Ab8…{10,}`, `sk-proj-…{20,}`, `whsec_…{20,}`, `nfp_…{20,}`
- **교훈(2026-07-04)**: 처음엔 접두사만 매치(`sk-ant-api|…`)했더니 SKILL.md 속
  패턴 설명 문자열이 자기 자신에 걸리는 오탐 발생 → 실키 길이·문자класс까지
  요구하는 정밀 패턴으로 수정. 검사 로직은 "매치 시 커밋 전 중단"이 원칙.

## 운영 시 주의

- 홈페이지 수정은 **`concept-c.html`만**. 로컬 `dist/`는 잔재(수정 금지, 배포와 무관).
- force push 금지. Netlify 빌드 실패 시 이전 배포가 유지됨(사이트 다운 아님) —
  로그: app.netlify.com/projects/anfilt-homepage/deploys
- Netlify 무료 플랜 빌드 분(월 300분) 소모 — 이 빌드는 수 초라 여유 큼.
