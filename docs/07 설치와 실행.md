# 07 설치와 실행

## 환경

- Windows 11 · Python 3.14 (3.10+ 필요)
- 프로젝트 경로: `C:\ANFILT_AI\Research Agent`

## 설치

```bash
pip install -r requirements.txt
```

주요 의존성: streamlit / anthropic / openai / google-genai / requests /
beautifulsoup4 / python-pptx / python-docx / openpyxl / pypdf /
google-api-python-client / google-auth / python-dotenv

## 환경변수 (.env)

`.env.example`을 `.env`로 복사 후 키 입력. **`.env`는 절대 공유·커밋 금지.**

| 변수 | 용도 | 현재 상태 (2026-07-04) |
|---|---|---|
| `ANTHROPIC_API_KEY` | Claude | 등록됨 — **크레딧 충전 필요** |
| `OPENAI_API_KEY` | GPT | 미등록 |
| `GOOGLE_API_KEY` | Gemini | ✅ 등록·검증 완료 |
| `ANTHROPIC_MODEL` 등 | 모델 오버라이드 (선택) | 기본값 사용 중 |
| `GAMMA_API_KEY` | Gamma 엔진 | 미등록 |
| `CANVA_ACCESS_TOKEN` | Canva 엔진 | 미등록 |
| `GOOGLE_SERVICE_ACCOUNT_FILE` | Google Slides 엔진 | 미등록 |

기본 모델: Claude `claude-opus-4-8` / GPT `gpt-5` / Gemini `gemini-2.5-pro`

## 실행

```bash
cd "C:\ANFILT_AI\Research Agent"
streamlit run app.py
```

개발용 미리보기 설정은 `.claude/launch.json`에 있음 (포트 8501).

## 실행당 비용 구조

LLM 호출 수 = **참여 LLM 수 × (1 + 토론 라운드) + 1(종합)**
- 예: 3개 LLM, 토론 1라운드 → 7회 호출
- 레퍼런스 자동 탐색은 별도 1회 (웹 검색 포함)
- 웹 검색은 프로바이더별 추가 요금 가능성 있음
- Gamma는 생성당 크레딧 차감

## 테스트

세션 스크래치에 테스트 스크립트가 있었음 (임시 폴더라 소멸 가능).
핵심 검증 항목은 [[08 작업 이력]]에 기록되어 있으므로, 회귀 테스트가
필요하면 다음을 다시 만들면 된다:
1. 보고서 3종 빌더 — 더미 REPORT_SCHEMA JSON으로 생성 + 재오픈 확인
2. `_length_plan` 1~20장 합계 = 목표 장수
3. `parse_json_loose` — 코드펜스/잡음/배열 케이스
4. `extract_file_text` — 형식별 샘플 파일 생성 후 추출
5. (키 필요) 프로바이더별 generate / 파이프라인 e2e
